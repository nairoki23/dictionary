# Excelでの編集の仕方
ExcelでTSVを書く方法はシンプルですが、**区切り形式と保存方法**を誤ると壊れます。実用手順だけ整理します。

---

## 基本手順

### 1. 列を作る

Excelでそのまま入力：

| A列      | B列   | C列  |
| ------- | ---- | --- |
| reading | word | pos |
| とうきょう   | 東京   | 地名  |
| じしょ     | 辞書   | 名詞  |

---

### 2. 保存

**重要：形式を選ぶ**

```
ファイル → 名前を付けて保存 → ファイルの種類
```

→ **「テキスト（タブ区切り）（*.txt）」を選択**

---

### 3. 拡張子変更

保存後：

```bash
sample.txt → sample.tsv
```
